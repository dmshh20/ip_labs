Область видимості, типізація й робота з консоллю

  • ⬅ На головну
  • Переглянути код JS
  • lpnu.ua
Мета

Закріпити роботу з консоллю Chrome DevTools: дослідити підняття змінних, відмінності між var/let/const, поведінку операцій приведення типів, незмінність об’єктів через Object.freeze() та створити невелику утиліту визначення типів.

Алгоритм виконання
  • Відкрийте сторінку в браузері, увімкніть DevTools (вкладка Console).
  • По черзі запускайте блоки із файлу lab6/js/main.js.
  • Занотовуйте передбачені результати та фактичні в консолі.
  • Порівняйте поведінку змінних різних областей видимості та операції приведення типів.
Запуск коду

Скрипт підключено з атрибутом defer, тому всі приклади виконуються одразу після завантаження DOM. Відкрийте консоль, щоб побачити коментарі та результати для кожного завдання. 

<script src="js/main.js" defer></script>

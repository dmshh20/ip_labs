Варіант 1 (непарні номери)

Скрипт lab8/js/main.js містить класичні функції, функціональні вирази, стрілкові та рекурсивні приклади, замикання та застосування колбеків до множини Set. Для тесту відкрийте DevTools → Console та викликайте методи об’єкта lab8.

Запуск прикладів
console.log(lab8.calculateSum(10));
console.log(lab8.multiply(3, 7));
console.log(lab8.power(2, 5));
console.log(lab8.harmonicSeries(5));
const triple = lab8.createMultiplier(3);
console.log(triple(9));
console.log(lab8.applyFunction(4, lab8.double));
lab8.processSet(new Set([1, 2, 3]), (x) => x * 10);
